# Character

## 角色屬性

| 參數名稱 | 英文縮寫 | 參數含義 |
| :--- | :--- | :--- |
| **生命值** | HP (Health Points) | 代表角色的生存能力，歸零時角色會倒下。 |
| **魔法值** | MP (Magic Points) | 施放魔法或技能時消耗的點數。 |
| **耐力** | STA (Stamina) | 部分行動會消耗耐力值，主要來自於體質（CON）。這讓高體質的角色除了血厚防高之外，還能擁有更長的持久戰能力。 |
| **物理攻擊力** | ATK (Attack) | 影響普攻和物理系技能造成的傷害。 |
| **魔法攻擊力** | MATK (Magic Attack) | 影響魔法系技能造成的傷害。 |
| **物理防禦力** | DEF (Defense) | 減少來自物理攻擊的傷害。 |
| **魔法防禦力** | MDEF (Magic Defense) | 減少來自魔法攻擊的傷害。 |
| **力量** | STR (Strength) | 主要影響物理攻擊力，部分遊戲也影響物理防禦力或負重。 |
| **智力** | INT (Intelligence) | 主要影響魔法攻擊力，部分遊戲也影響魔法值或魔法防禦力。 |
| **敏捷** | AGI (Agility) | 主要影響攻擊速度、迴避率和行動順序，是影響戰鬥節奏的關鍵。 |
| **體質** | CON (Constitution) | 主要影響生命值，部分遊戲也影響物理防禦力或負重。 |
| **幸運** | LUK (Luck) | 影響暴擊率、掉寶率或某些特殊事件的成功率。 |
| **準確度** | ACC (Accuracy) | 影響攻擊命中率。 |
| **迴避率** | EVA (Evasion) | 影響躲避攻擊的機率。 |
| **暴擊率** | CRT (Critical Rate) | 觸發暴擊的機率，暴擊通常會造成更高的傷害。 |

## 屬性計算

* **生命值（HP）= CON × 20**
* **魔法值（MP）= INT × 15**

---

* **物理攻擊力（ATK）= STR × 2**
* **魔法攻擊力（MATK）= INT × 2**

---

* **物理防禦力（DEF）= CON × 2**
* **魔法防禦力（MDEF）= INT × 2**

---

* **準確度（ACC）= AGI × 2**
* **迴避率（EVA）= AGI × 2 + LUK**

---

* **暴擊率（CRT）= LUK × 1 + AGI × 0.5**

## 戰鬥計算

你的建議很好。百分比減傷能讓防禦在遊戲後期依然有價值，不會因為敵人的傷害數值膨脹而變得無力。同時，調整命中和迴避的影響力，能讓你對每個招式的命中率有更精確的控制，這能讓戰術設計更豐富。

這是根據你的新想法，重新設計的公式：

### 傷害與防禦計算

**傷害計算公式：**
`最終傷害 = 攻擊力 × (1 - 減傷率)`

* **物理傷害：** `最終物理傷害 = ATK × (1 - 物理減傷率)`
* **魔法傷害：** `最終魔法傷害 = MATK × (1 - 魔法減傷率)`

**減傷率計算：**

* **物理減傷率 = DEF / (DEF + 100)**
* **魔法減傷率 = MDEF / (MDEF + 100)**

**設計考量：**
這個公式會讓減傷率介於 **0%** 到 **100%** 之間，且隨著防禦力的增加，每點防禦力帶來的收益會逐漸遞減。
* 當 **DEF** 為 100 時，減傷率為 **100/(100+100) = 50%**。
* 當 **DEF** 為 300 時，減傷率為 **300/(300+100) = 75%**。
* 當 **DEF** 為 900 時，減傷率為 **900/(900+100) = 90%**。

---

### 命中、迴避與暴擊計算

我們將讓命中和迴避的影響變小，將它們視為一個小幅度的修正值，而不是決定性的因素。

* **基礎命中率**：你將為每個招式設定一個**基礎命中率**，例如物理攻擊的基礎命中率為 95%，某些特殊技能為 70%。
* **命中率修正值**：`命中率修正值 = (你的 ACC - 敵人的 EVA) × 0.2%`
    * 為了避免修正值過大，這裡設定了一個很小的係數 `0.2%`。
    * 你的 **ACC** 比敵人的 **EVA** 高 10 點，修正值就是 **(10) × 0.2% = 2%**。這意味著你的攻擊命中率會增加 2%。

**最終命中率 = 招式基礎命中率 + 命中率修正值**

* **最終迴避率**：`最終迴避率 = 招式基礎迴避率 + 迴避率修正值`
    * 同理，你可以為每個招式設定一個基礎迴避率，然後加上修正值。這會讓角色的敏捷和幸運值不會完全決定是否被擊中。

**暴擊率計算：**
* **最終暴擊率 = 招式基礎暴擊率 + 暴擊率修正值**
    * 暴擊率修正值可以沿用我們之前的公式：`暴擊率修正值 = (LUK × 1 + AGI × 0.5) × 0.5%`。
    * 這能讓暴擊率的影響也保持在一個可控的範圍內。

**設計考量：**
這種方式賦予了你更大的自由度，你可以透過招式設計來平衡遊戲，而不是完全依賴角色的屬性。例如：
* **一個高傷害但命中率低的招式**：基礎命中率設為 60%，讓高 **AGI** 的角色能透過修正值來提高命中率。
* **一個防禦性招式**：可以讓它的基礎迴避率非常高，讓所有角色都能有效閃避攻擊，而高 **AGI** 的角色則能讓成功率更高。

## 角色範本

根據 25 點的總點數限制，設計了兩個風格截然不同的角色範本。

### 範本一：狂戰士 (Warrior)
這是一個專注於物理輸出和生存能力的近戰角色。他放棄了魔法和暴擊，將所有點數都投入到力量和體質上，以此換取極高的物理傷害與強韌的生存力。

---

| 參數名稱 | 屬性點分配 |
| :--- | :--- |
| **力量 (STR)** | 12 |
| **智力 (INT)** | 0 |
| **敏捷 (AGI)** | 3 |
| **體質 (CON)** | 10 |
| **幸運 (LUK)** | 0 |
| **總計** | 25 |

#### 戰鬥屬性計算

* **生命值 (HP)** = CON (10) × 20 = **200**
* **魔法值 (MP)** = INT (0) × 15 = **0**
* **物理攻擊力 (ATK)** = STR (12) × 2 = **24**
* **魔法攻擊力 (MATK)** = INT (0) × 2 = **0**
* **物理防禦力 (DEF)** = CON (10) × 2 = **20**
* **魔法防禦力 (MDEF)** = INT (0) × 2 = **0**
* **準確度 (ACC)** = AGI (3) × 2 = **6**
* **迴避率 (EVA)** = AGI (3) × 2 + LUK (0) = **6**
* **暴擊率 (CRT)** = LUK (0) × 1 + AGI (3) × 0.5 = **1.5%**

**角色定位：**
**狂戰士**擁有極高的 **HP** 和 **DEF**，這讓他在面對物理敵人時幾乎是不可撼動的。他的 **ATK** 也很可觀，可以穩定地造成大量傷害。但他的致命弱點在於魔法攻擊，由於 **INT** 為零，他既沒有 **MP** 也沒有 **MDEF**，面對魔法敵人將會非常脆弱。

### 範本二：秘法刺客 (Arcane Rogue)
這是一個專注於敏捷、暴擊與魔法傷害的混合型角色。他犧牲了大部分的物理防禦和血量，但換來了高爆發、高機動性和強大的魔法能力，可以在戰場上靈活穿梭。

---

| 參數名稱 | 屬性點分配 |
| :--- | :--- |
| **力量 (STR)** | 2 |
| **智力 (INT)** | 8 |
| **敏捷 (AGI)** | 10 |
| **體質 (CON)** | 2 |
| **幸運 (LUK)** | 3 |
| **總計** | 25 |

#### 戰鬥屬性計算

* **生命值 (HP)** = CON (2) × 20 = **40**
* **魔法值 (MP)** = INT (8) × 15 = **120**
* **物理攻擊力 (ATK)** = STR (2) × 2 = **4**
* **魔法攻擊力 (MATK)** = INT (8) × 2 = **16**
* **物理防禦力 (DEF)** = CON (2) × 2 = **4**
* **魔法防禦力 (MDEF)** = INT (8) × 2 = **16**
* **準確度 (ACC)** = AGI (10) × 2 = **20**
* **迴避率 (EVA)** = AGI (10) × 2 + LUK (3) = **23**
* **暴擊率 (CRT)** = LUK (3) × 1 + AGI (10) × 0.5 = **8%**

**角色定位：**
**秘法刺客**的生命值和物理防禦極低，意味著他無法承受任何正面的物理攻擊。但他擁有極高的 **ACC** 和 **EVA**，可以確保攻擊命中並有機會閃避敵人的攻擊。配合高 **MATK** 和 **MP**，他可以靈活運用魔法技能進行遠程攻擊。雖然 **CRT** 不算特別高，但這 8% 的暴擊率在關鍵時刻足以決定勝負。

這兩個範本展示了在 25 點的限制下，玩家必須做出艱難的取捨。這將引導玩家思考：「我願意為了高傷害放棄防禦嗎？」「我願意為了生存犧牲魔法嗎？」這種決策過程正是這類遊戲的樂趣所在。
